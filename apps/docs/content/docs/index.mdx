Porkbun is a set of TypeScript tools for the [Porkbun](https://porkbun.com) domain registrar:

- **`porkbun-sdk`** — SDK to integrate Porkbun into your applications
- **`porkbun-cli`** — CLI to manage your domains from the terminal

## Installation

<Tabs items={["SDK TypeScript", "CLI"]}>
  <Tab value="SDK TypeScript">
    ```bash title="Terminal"
    npm install porkbun-sdk
    ```
  </Tab>
  <Tab value="CLI">
    ```bash title="Terminal"
    npm install -g porkbun-cli
    ```
  </Tab>
</Tabs>

## Quick start

<Tabs items={["SDK TypeScript", "CLI"]}>
  <Tab value="SDK TypeScript">
    <Steps>
      <Step>
        ### Set your API keys

        Get your keys from [porkbun.com/account/api](https://porkbun.com/account/api).

        ```bash title="Terminal"
        export PORKBUN_API_KEY=pk1_xxxxx
        export PORKBUN_API_SECRET=sk1_xxxxx
        ```
      </Step>

      <Step>
        ### Use the SDK

        ```ts title="app.ts"
        import { PorkbunClient } from "porkbun-sdk";

        const client = new PorkbunClient({
          apikey: process.env.PORKBUN_API_KEY!,
          secretapikey: process.env.PORKBUN_API_SECRET!,
        });

        const ping = await client.ping();
        const domains = await client.domain.listAll();
        const records = await client.dns.retrieve("example.com");
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab value="CLI">
    <Steps>
      <Step>
        ### Set your API keys

        ```bash title="Terminal"
        export PORKBUN_API_KEY=pk1_xxxxx
        export PORKBUN_API_SECRET=sk1_xxxxx
        ```
      </Step>

      <Step>
        ### Use the CLI

        ```bash title="Terminal"
        porkbun ping
        porkbun domain list --format table
        porkbun dns list example.com
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
